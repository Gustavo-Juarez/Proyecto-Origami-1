CSS significa Cascading Style Sheets.

  • CSS es el lenguaje que usamos para dar estilo a un documento HTML.
  • CSS describe como los elementos HTML deben ser mostrados.
  • CSS ahorra mucho trabajo. Puede controlar el diseño de múltiples páginas web todas a la vez.
